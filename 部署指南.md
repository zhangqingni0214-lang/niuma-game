## 二、三条上线路径（推荐 Vercel）

### 🟢 路径 A：Vercel 拖拽（推荐 · 30 秒出 URL）

**适用**：要 https URL、要海外可访问、不想折腾

**步骤**：

1. 打开 [vercel.com](https://vercel.com)，用 GitHub / Google 账号登录
2. 点右上角 **Add New → Project**
3. 选 **Browse → Upload**（不用连 git，直接传文件夹）
4. 把整个 `/Users/a58/Desktop/Claude本地/牛马游戏/` 文件夹**拖进去**
5. 等 10 秒，自动给你一个 URL，类似：
   `https://niuma-company-abc123.vercel.app`
6. 点 **Settings → Domains** 可以改子域名为 `niuma.vercel.app`（如果没被占）

**优点**：
- 免费，永久
- 自带 HTTPS + 全球 CDN
- 支持自定义域名（买个 ¥50/年的 `.com` 接上即可）

**缺点**：
- 国内访问偶尔慢（但可访问）
- 默认子域名 `.vercel.app` 在小部分企业网会被墙

---

### 🟡 路径 B：Netlify Drop（备选 · 同样 30 秒）

**适用**：和 Vercel 二选一

**步骤**：

1. 打开 [app.netlify.com/drop](https://app.netlify.com/drop)
2. 直接把整个项目文件夹**拖到页面里**（不用注册也能传，但建议注册以便管理）
3. 给你一个 URL：`https://random-name-12345.netlify.app`
4. 注册后可改子域名

跟 Vercel 体验几乎一样，全凭习惯选。

---

### 🔴 路径 C：腾讯云 / 阿里云 OSS（国内可访问最稳）

**适用**：目标用户在国内、希望访问速度最稳、不在乎花钱

**步骤**：

1. 开通 **腾讯云 COS** 或 **阿里云 OSS**（按量付费，每月 ¥1–5）
2. 创建一个 **Bucket**，权限设为"公开读"
3. 用桌面客户端（COSBrowser / OSS Browser）上传整个项目文件夹
4. 开启"静态网站"功能，指定 index.html 为首页
5. 拿到 URL 类似 `https://niuma.oss-cn-shanghai.aliyuncs.com/index.html`
6. **必须备案 + 绑定自有域名**才能用 `niuma.com` 这样的短链

**优点**：国内访问最稳、合规
**缺点**：
- 域名要备案（个人备案 1–3 周）
- 没自有域名只能用难看的长链
- 要花点钱（每月 ¥1–5）

---

### 🆓 路径 D：GitHub Pages（最便宜，但要懂 git）

**适用**：你/团队会用 git

**步骤**：

1. GitHub 新建 repo（设为 public，因为 Pages 免费版要求 public）
2. 把项目代码 push 上去
3. Settings → Pages → Source 选 `main` 分支
4. 等 1 分钟，URL：`https://你的用户名.github.io/niuma-company`

**优点**：完全免费、版本可追溯
**缺点**：国内访问不稳定，要会 git

## 三、推荐路径

按你的实际情况：

| 你的情况 | 推荐 |

## 五、上线前优化建议（可选）

### 5.1 图片体积

`assets/` 目录里几个 PNG 接近 2.5MB，加起来 15MB+：

```
horse_bg.png      2.5 MB
ox_bg.png         2.4 MB
menu_bg.png       2.4 MB
horse_4x5.png     1.9 MB
ox_4x5.png        1.5 MB
office_wide.png   1.3 MB
_mockup_reference.png  1.4 MB  ← 这个不该部署，是设计稿
```

首次加载会偏慢（特别是移动 4G）。

**改进建议（部署前可做）**：

1. **删 `_mockup_reference.png`**（设计稿不用上线）
2. **压缩 PNG**：用 [tinypng.com](https://tinypng.com) 拖进去，体积砍 60-70% 没问题
3. **字体二选一**：`vonwaon-16px.ttf` 和 `凤凰点阵体VonwaonBitmap16px.ttf` 是同一个文件，留一个

做完总体积可以从 18MB 降到 6–8MB。

如果你想现在就做，告诉我我帮你删 mockup_reference 和重复字体（压缩 PNG 是图形工具的事，我做不了）。

### 5.2 SEO / 分享卡

我已经加了 `og:title` / `og:description`，但还可以加一个 `og:image`（朋友圈分享会显示的封面图）：

```html
<meta property="og:image" content="https://你的域名/assets/menu_bg.png" />
```

等你拿到 URL 后我帮你加。

### 5.3 favicon

浏览器标签栏现在是空白图标。要做的话：

1. 去 [emojico.io](https://emojico.io/) 输入 🐴 生成 favicon.ico
2. 放到根目录
3. index.html 加 `<link rel="icon" href="favicon.ico" />`

3 分钟搞定。
